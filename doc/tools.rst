Tools
=====
This section describes additional tools that BAF provides. Most of them
have been added to support old-style FCM config files, and LFRic-specific
features.

.. _fcm_extract:

FcmExtract
----------
The FcmExtract helps in converting old-style FCM extract configuration files
into Fab based ``Exclude`` and ``Include`` objects.
